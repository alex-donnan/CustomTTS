How to Play the Lute

Select a BPM and add notes in the boxes below that will be played. A note can have 4 parts:

  • Note: a-b and r, lowercase. Another b means flat. An r is a rest.
  • Octave: Optional: 1-5, defaults to 4.
  • Accent: Optional: *, will try to accent that note.
  • Duration: 1-4, duration in beats. Cannot be outside this range.

The format for this will be NoteOctaveAccent.Duration, some examples:

  • ab.1 - a flat, octave 4, 1 beat
  • d2.4 - d, octave 2, 4 beats
  • db3*.2 - d flat, octave 3, accented, 2 beats

Adding a new line of notes will add another voice. The last voice is always the bass line, so be sure to write that last.

The character limit on Twitch is 500 characters, please be mindful of this! It could break the generator.

The box at the bottom will contain the message you have to use in chat. Make sure you Cheer first!
